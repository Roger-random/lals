    Queue of Interesting Things I Want To Investigate Later#

    Items on this list are only tangentially related to trains and don’t really fit elsewhere on this site. Items of interest with direct train applicability usually get their own page.

    AkiraOS Built on Zephyr RTOS for running WebAssembly (WASM) bytecode in a sandbox on microcontrollers in a container-like fashion. Allows application code in WASM to be dynamically updated without reflashing the entire microcontroller.

    Armbian Create Ubuntu or Debian images for flashing on single board computers.

    NiceGUI Python based system for developing a browser interface

    RoSys Robotics infrastructure project with concepts similiar to ROS but focuses on Python instead of ROS language agnostic overhead. Uses NiceGUI for browser interface and asyncio for cooperative multitasking.
